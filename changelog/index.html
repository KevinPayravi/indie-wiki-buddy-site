3.13.5

April 11, 2025 April 10, 2025
  • Fix for Brave Search result filtering.
    • Starting around April 9, if Indie Wiki Buddy search engine filtering was enabled for Brave, the results would flash for a second and then disappear, leaving a blank page.
    • The issue appears to have started after Brave Search was updated with JavaScript code that expects a specific DOM structure. When search engine filtering is enabled, IWB will insert a few HTML elements (e.g. the blue button that points users to independent wikis). These elements conflicted with Brave's JavaScript, causing the page to flash and then disappear when Brave would try to refresh the DOM. This has been fixed by changing the position of the DOM elements Indie Wiki Buddy inserts into the page, avoiding conflict with Brave's code.
    • Apologies for any inconvenience for Brave Search users, and thank you to everyone who kindly reported the issue and provided details to help debug!
  • Improved light/dark mode color handling for elements injected into search engine results (redirect buttons, text, etc.)

    • This feature is to help surface the original, content-rich independent wiki results.

    • This feature is currently only available for Google, as it can be complicated to maintain. Other search engine may be supported in the future.
    • Thank you to the Weird Gloop team for contributing this new feature!

This is the new default functionality, and is especially helpful for finding new wikis that don't rank well in search engine results yet.     • A bug has been fixed where the "set all" buttons could conflict with other selections.

  • The Chromium bug banner has been updated. Chromium bug 1271154 has been resolved, but there is a new bug 1498035 that can also cause the extension to stop working after an update. If a user faces issues this issue, they now just need to go to their browser settings and toggle the extension off and on again, instead of re-installing the extension entirely.
  • The Opera and Chromium notices are now dismissible by the user.
